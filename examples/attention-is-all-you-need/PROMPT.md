# The prompt this project was built from

> Make me a short video that explains the attention mechanism from "Attention
> Is All You Need" to software engineers. About a minute. Two hosts, dark
> theme. The key example should be the sentence "the animal didn't cross the
> street because it was too tired".

Everything in `reel.config.mjs` — the five-scene arc (hook → the problem →
the idea → the worked example → why it matters), the questioner/explainer
casting, the pacing, and the theme — was decided by narova from this prompt.
