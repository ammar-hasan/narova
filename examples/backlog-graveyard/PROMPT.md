# The prompt this project was built from

> Punchy 30-second vertical reel for engineering social: the hot take "your
> backlog is a graveyard". Two hosts, fast back-and-forth, big bold numbers,
> warm sunset palette — nothing corporate.

Deliberately different from the other examples: `9:16`, reel pacing (tight
gaps, tempo 1.22), `.stat` / `.verdicts` / `.stepper` / `.bigquote` layouts
instead of centered title cards, and a custom `theme.css` signature.
